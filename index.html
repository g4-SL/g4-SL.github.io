Child First Reporting System

Worked closely with a team of student developers to design and construct a mobile application that improves the accessibility of reporting child abuse cases to the appropriate child protection services. Planned and discussed ideas for the design flow of the application to ease the social workers’ interview with the abused child and handling and retrieval of all cases. Implemented and designed the front end interface of the application using frameworks such as Phonegap, AngularJS and Bootstrap. Contributed as the team’s coordinator to delegate work among the team and communicate effectively with all developers to keep track with the progress of the project

  • Client: Kota Takaoka Research Lab
  • Date: September 2014 - April 2015
  • Language/Tool: Javascript, HTML/CSS, AngularJS

Penetration Testing on UBC Okanagan Athletics Database

Discussed and analyzed the security of UBC-O Athletics registration website with the system owner and developers. Performed security tests such as replay attack, SQL injection, cross-site scripting, command injection, session hijacking, and man-in-the-middle attack on the website to determine the level of security the website provides to all related stakeholders. Reported all findings to the system owner and brainstormed for possible security solutions based on the findings of the research

  • Client: UBC Okanagan
  • Date: September 2014 - December 2014
  • Language/Tool: Javascript, TamperData, Webscarab
