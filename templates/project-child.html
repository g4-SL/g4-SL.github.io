Child First Reporting System

Child First is a website that serves to improve the accessibility of reporting child abuse cases to the appropriate child protection services. Together with Kota Takaoka Reserach Lab, the website was built to give further support to professional social workers at Japan and Vancouver to simplify all child abuse reporting processes.

The website includes a login system to separate both administrators and professional social workers accounts, a dual language support (English and Japanese), and a risk assessment sheet form that is to be submitted for any child abuse reports.

During this project, I was responsible for designing and implementing the front-end interface and also linking the website to an external server for database storage. Child First is built using AngularJS (Javascript, HTML/CSS) and uses PHP as a communication language between the front-end and back-end.
