Olympia Prototype Design

Olympia is a prototype mobile application that targets audiences who are keen on keeping themselves updated with the latest news on all ongoing Olympic events. Olympia is designed as an application with a simple and intuitive interface so that users can surf around the application without hindering their busy and active lifestyle. The application focuses more on user customization to deliver Olympic news that the user is interested in to better serve the user's purpose on using the application.

Together with four other students, Olympia was designed under guidance of Professor Sid Fels for ECE 418 (Human Computer Interfaces).



For the purpose of user customization on Olympia, the application will prompt the user to select their allegiance on their supported country, interested sports and athletes. The home screen of the application will display relevant Olympic news according to these selected choices.

Olympia offers user to check current medal count, schedule for upcoming Olympic events, live event updates, and a chatroom with other users.
